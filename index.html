Guia Completo de Tags HTML

Um guia exaustivo para as tags mais importantes e usadas do HTML5.

Observação:

Para envelopar uma tag em um componente já criado no VS Code, selecione os elementos, pressione Control + Shift + P, digite Wrap with Abbreviation e, em seguida, escreva a tag desejada.

1. Estrutura Base do Documento

Essas tags formam o esqueleto de qualquer página web.

<!DOCTYPE html>

É a primeira linha de código e informa ao navegador que o documento é um arquivo HTML5.

<!DOCTYPE html>

<html>

É o elemento raiz, ou seja, o contêiner principal que engloba toda a página.

<html></html>

<head>

Uma seção que contém metadados (informações sobre a página) que não são visíveis para o usuário. Aqui, você define o título, a codificação de caracteres, links para arquivos CSS e JavaScript, etc.

<head>
  <meta charset="UTF-8">
  <title>Meu Portfólio de Fotografia</title>
  <link rel="stylesheet" href="estilos.css">
</head>

<body>

Contém todo o conteúdo visível da página, como títulos, parágrafos, imagens, links e vídeos.

<body>
  <h1>Bem-vindo ao meu Portfólio!</h1>
  <p>Confira minhas melhores fotos na galeria abaixo.</p>
</body>

2. Estrutura Semântica e Seções

Essas tags organizam o conteúdo de uma forma que faz sentido não só para você, mas também para os navegadores e para leitores de tela usados por pessoas com deficiência visual.

<header>

Representa o cabeçalho de uma seção ou da página, geralmente contendo o título principal, o logo do site e a navegação.

<header>...</header>

<nav>

Define uma seção que contém links de navegação.

<nav>
  <ul>
    <li><a href="#sobre">Sobre</a></li>
    <li><a href="#servicos">Serviços</a></li>
    <li><a href="#contato">Contato</a></li>
  </ul>
</nav>

<main>

Representa o conteúdo principal e único de uma página. Deve haver apenas uma tag <main> por documento.

<main>...</main>

<article>

Define um conteúdo independente e autocontido, como um post de blog, uma notícia ou um comentário. Um <article> pode ser republicado em outros sites sem perder o sentido.

<article>...</article>

<section>

Representa uma seção genérica de conteúdo em um documento. É usada para agrupar conteúdos relacionados.

<section>...</section>

<aside>

Representa um conteúdo que é relacionado, mas separado, como uma barra lateral, uma caixa de curiosidades ou uma lista de links relacionados.

<aside>...</aside>

<footer>

Representa o rodapé de uma seção ou da página, contendo informações como direitos autorais, links para redes sociais e informações de contato.

<footer>...</footer>

<figure> e <figcaption>

A tag <figure> é usada para agrupar conteúdo autocontido (geralmente uma imagem, um diagrama ou um gráfico) e a tag <figcaption> fornece uma legenda para ele.

<figure>
  <img src="grafico-vendas.png" alt="Gráfico de Vendas">
  <figcaption>Gráfico mostrando o aumento de vendas.</figcaption>
</figure>

3. Conteúdo em Linha e Formatação de Texto

Essas tags são usadas para formatar o texto e dar significado a ele.

<h1> a <h6>

Representam títulos de diferentes níveis de importância. <h1> é o título principal da página, e os outros servem para subtítulos.

<h1>Título principal</h1>
<h2>Subtítulo de uma seção</h2>

<p>

Define um parágrafo de texto.

<p>Este é um parágrafo de texto.</p>

<strong> e <b>

<strong> indica forte importância (semântica). <b> é apenas para dar um destaque visual em negrito, sem adicionar importância.

<p><strong>Atenção:</strong> Este é um aviso importante.</p>

<em> e <i>

<em> enfatiza um texto (semântica). <i> é para um texto em itálico com uma conotação diferente da ênfase (ex: um termo técnico ou um nome de livro).

<p>O prazo de entrega é de <em>dois dias úteis</em>.</p>

<span> e <div>

Ambos são contêineres genéricos. <span> é em linha (não quebra a linha) e <div> é um bloco (cria uma nova linha). Eles são usados para agrupar elementos e aplicar estilos.

<div>
  <p>Seção de Clientes com <span class="text-red-500">desconto</span>.</p>
</div>

<pre>

Formata o texto de forma pré-formatada, preservando os espaços e quebras de linha que foram escritos no código.

<pre>
  Função:
    if (condicao) {
      return true;
    }
</pre>

<blockquote> e <q>

<blockquote> para citações longas. <q> para citações curtas.

<blockquote>"A persistência é o caminho do êxito."</blockquote>
Ele disse: <q>olá</q>.

<br> e <hr>

<br> causa uma quebra de linha. <hr> cria uma linha horizontal.

Texto aqui<br>e mais texto.<hr>

<sub> e <sup>

<sub> (subscrito) para texto abaixo. <sup> (sobrescrito) para texto acima.

H<sub>2</sub>O e 20<sup>3</sup>

<del> e <ins>

<del> indica texto riscado (deletado). <ins> indica texto sublinhado (inserido).

Preço: <del>R$50</del> <ins>R$40</ins>

4. Links, Listas e Tabelas

Essas tags são essenciais para organizar o conteúdo de forma hierárquica e interligada.

<a>

Cria um link que direciona o usuário para outra página web, um arquivo para download ou uma seção específica dentro da mesma página.

<a href="https://google.com" target="_blank" rel="external">Acessar o Google</a>
<a href="/downloads/arquivo.pdf" download="nome-do-arquivo">Baixar PDF</a>

<ul>, <ol> e <li>

A tag <ul> cria uma lista não ordenada (com marcadores), <ol> cria uma lista ordenada (numerada) e <li> é um item de lista.

<h3>Ingredientes</h3>
<ul>
  <li>Café</li>
</ul>
<h3>Modo de Preparo</h3>
<ol>
  <li>Pré-aqueça o forno.</li>
</ol>

<table> e relacionados

Estruturas para criar tabelas com cabeçalho (<thead>), corpo (<tbody>), rodapé (<tfoot>), linhas (<tr>), e células de cabeçalho (<th>) e de dados (<td>).

<table>
  <thead><tr><th>Produto</th></tr></thead>
  <tbody><tr><td>Café</td></tr></tbody>
</table>

<caption>, <colgroup>, <col>

<caption> adiciona legenda à tabela. <colgroup> e <col> estilizam colunas.

<table>
  <caption>Tabela de Preços</caption>
  <colgroup><col span="2" style="background-color:gray"></colgroup>
</table>

5. Formulários e Interatividade

Essas tags permitem que os usuários interajam com a página, enviando dados para um servidor.

<form>

Agrupa todos os elementos de um formulário (campos de texto, botões, etc.) para que os dados sejam enviados juntos.

<form action="/enviar-contato" method="post"></form>

<label> e <input>

Define um rótulo para um campo de formulário e cria um campo de entrada de dados, com vários tipos (text, email, password, etc.).

<label for="email">Seu E-mail:</label>
<input type="email" id="email" name="email">

<button>

Cria um botão clicável.

<button type="submit">Enviar Mensagem</button>

<textarea>

Define uma área de texto multi-linha, ideal para mensagens longas ou comentários.

<textarea id="mensagem" rows="4"></textarea>

<select> e <option>

<select> cria uma lista suspensa (dropdown) e <option> define as opções que o usuário pode escolher.

<select id="pais">
  <option value="br">Brasil</option>
</select>

<fieldset> e <legend>

<fieldset> agrupa elementos de um formulário. <legend> define a legenda.

<fieldset>
  <legend>Contato</legend>
</fieldset>

<datalist>

Cria uma lista de opções para um campo de input.

<input list="browsers">
<datalist id="browsers">
  <option value="Chrome"></option>
  <option value="Firefox"></option>
</datalist>

6. Multimídia e Canvas

Essas tags permitem que você adicione imagens, vídeos, áudios e gráficos à sua página, tornando-a mais rica e interativa.

<img>

Incorpora uma imagem em seu documento HTML.

<img src="foto.jpg" alt="Minha foto">

<video> e <audio>

Incorpora vídeo e áudio.

<video controls width="500" height="300" poster="thumbnail.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Seu navegador não suporta a tag de vídeo.
</video>
<audio src="musica.mp3" controls loop></audio>

<source>

Define a fonte de um arquivo de mídia para <video> ou <audio>.

<video controls>
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta vídeos.
</video>

<picture>

Permite exibir imagens diferentes para diferentes tamanhos de tela.

<picture>
  <source srcset="imagem-mobile.jpg" media="(max-width: 600px)" />
  <img src="imagem-desktop.jpg" alt="Descrição da imagem" />
</picture>

<canvas>

Uma "tela" para desenhar gráficos e animações com JavaScript.

<canvas id="meuGrafico"></canvas>

<svg>

Usado para criar gráficos vetoriais.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" fill="red" />
</svg>

7. Tags de Interatividade e Conteúdo Dinâmico

Essas tags trazem recursos de interface mais avançados, que eram feitos antes com JavaScript, mas agora têm suporte nativo no HTML.

<dialog>

Usado para criar janelas de diálogo, pop-ups ou modais. É a forma semântica e acessível de criar esses elementos.

<button onclick="modal.showModal()">Abrir</button>
<dialog id="modal">
  <p>Conteúdo do modal.</p>
</dialog>

<details> e <summary>

Cria um widget de divulgação onde o usuário pode abrir ou fechar o conteúdo, e <summary> define o título que fica sempre visível.

<details>
  <summary>O que é HTML?</summary>
  <p>Linguagem de Marcação de Hipertexto.</p>
</details>

<template> e <slot>

<template> armazena código HTML que pode ser usado com JavaScript. <slot> é um placeholder para injeção de HTML em componentes.

<template id="card-template">...</template>

<output>

Usado para exibir o resultado de um cálculo ou ação do usuário.

<form oninput="total.value=parseInt(a.value)+parseInt(b.value)">
  <input type="number" id="a"> + <input type="number" id="b"> = <output name="total"></output>
</form>

<progress> e <meter>

<progress> exibe o progresso de uma tarefa. <meter> representa um valor em um intervalo.

<progress value="70" max="100">70%</progress>
<meter value="0.75"></meter>

8. Tags de Script e Estilo

Essas tags são usadas para ligar o HTML ao JavaScript e ao CSS.

<script>

Usado para incorporar código JavaScript.

<p id="paragrafo">Clique para mudar o texto.</p>
<script>
  document.getElementById("paragrafo").onclick = function() {
    this.innerHTML = "O texto mudou!";
  };
</script>

<style>

Adiciona código CSS diretamente no documento.

<style>
  button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
  }
</style>
<button>Botão Estilizado</button>

9. Outras Tags e Atributos

Tags e atributos menos comuns, mas ainda válidos e úteis.

<wbr>

Quebra uma palavra longa se necessário.

<p>A url do site é: http://exemplo<wbr>de-site-muito-longo.com</p>

<bdo>

Sobrescreve a direção do texto.

<p><bdo dir="rtl">Este texto vai da direita para a esquerda.</bdo></p>

<cite>

Define o título de um trabalho.

<p>O livro <cite>O Senhor dos Anéis</cite> foi escrito por J.R.R. Tolkien.</p>

<mark>

Marca ou destaca um texto.

<p>A resposta correta é: <mark>HTML</mark>.</p>

<u>

Sublinha o texto.

<p><u>Sublinhado</u>.</p>

<code>

Exibe código em linha.

<p>Para exibir uma mensagem, use o comando <code>alert()</code>.</p>

<time>

Representa uma data e/ou hora.

<p>O evento começa em <time datetime="2025-10-12T19:00">12 de Outubro, às 19:00h</time>.</p>

<s>

Indica que o conteúdo não é mais relevante.

<p>Preço antigo: <s>R$50</s>.</p>

<small>

Representa texto de menor importância.

<p>Este é o conteúdo principal. <small>Aviso legal: todos os direitos reservados.</small></p>

<dfn>

Define um termo que será introduzido ou definido.

<p><dfn>HTML</dfn> é a linguagem de marcação para a web.</p>
